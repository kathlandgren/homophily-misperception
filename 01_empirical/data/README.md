# Empirical data

The data files in this directory are not tracked in git.

They come from the following source:

> Sparkman, Gregg, Nathan Geiger, and Elke U. Weber. "Americans experience a false social reality by underestimating popular climate policy support by nearly half." *Nature Communications* 13.1 (2022): 4779.
> https://doi.org/10.1038/s41467-022-32412-y

Please obtain the data from the paper's supplementary materials or contact the authors, then place the files in this directory before running the empirical notebooks.
